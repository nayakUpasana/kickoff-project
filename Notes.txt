If pyscopg2 doesn't get installed by the following command
>> pip install psycopg2
Use
>> pip install psycopg2-binary

To install all the requirements directly from the terminal to the requirements file:
>> pip freeze > code/requirements.txt

To upgrade an already package
>> pip install --upgrade Flask-JWT-Extended
